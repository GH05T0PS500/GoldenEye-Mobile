GoldenEye Mobile
================

GoldenEye is an Android app for SECURITY TESTING PURPOSES ONLY!

GoldenEye is a Layer 7 DoS Test Tool. This is a port to Android of the GoldenEye tool (https://github.com/jseidl/GoldenEye)

NO ROOT REQUIRED!
